# I-Like-Baby-server
아이조아(I-Like-Baby) 서버


### Commit Message Convention
|Tag|Description|
|:-:|:-:|
|Feat|새로운 기능 추가|
|Fix|단순 버그 수정|
|Style|코드 포맷 변경, 세미콜론 누락, 필요없는 패키지 제거 등|
|!HOTFIX|치명적인 버그 수정|
|Refactor|코드 리팩토링|
|Comment|주석 추가 및 수정, 삭제|
|Test|테스트 코드 추가 테스트 코드 변경|
|Chore|빌드 도구(Gradle) 수정, 패키지 추가|
|Rename|파일,폴더명 수정|
|Remove|파일 삭제|
|Config|설정 파일 수정|
|Move|파일 위치 변경|
|Docs|문서 수정(README.MD 등)|
